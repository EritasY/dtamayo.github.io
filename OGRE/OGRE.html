OGRE

OGRE (Orbital GRaphics Environment) is an open-source software tool that reads in the output from commonly used orbital integrators, and then allows the user to both interactively view the simulation, and generate movies of the integration. The movie files are in .mp4 format and can thus be easily inserted into professional, teaching, or outreach presentations.

OGRE was primarily written by Rob Douglas and Heming Ge--I am responsible for regularly introducing new bugs. Learn more at About OGRE. You can get the source code or a self-contained executable on the Download page. We have spent a LONG time on the Source Code Documentation (go to related pages) so that as people start using OGRE and develop their own personal needs, they can contribute to the code. Also check out the User's Guide.

Please let me know if you find any bugs, as well as any features you would like to see added!
