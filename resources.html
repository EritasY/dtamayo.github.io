Resources

This page collects several resources to help get you started. It doesn't make sense to recreate any of the great linux tutorials available online. If you've never used linux before (or need a refresher), start by going through all of Section 2 of the linked Basic linux commands and practice them on one of our workstations. The rest of the links cover things that are harder to sort out for the first time, and provide solutions to some of the peculiarities with our particular workstation setup. Hopefully they give you the right terms to google if you want to dig deeper, and cover some of the most useful options for those commands (out of a laundry list that can be quite overwhelming when you're googling a command for the first time). The relevant guides for working on your laptop remotely are written for both Mac and Windows. Please do ask questions if you get stuck--that will help us improve these guides for others!
